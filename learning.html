M1

M2
⚡ Build Velocity
Month 2 · Weeks 5–8
Projects Begin
Brand Publish 8 posts total. Start writing about what you're learning (Python, AI). Add GitHub project links in posts.
Python Advanced Python: decorators, generators, async. Build and publish a FastAPI microservice. Connect to a real API or DB.
AI Data Science 2 + start RAG fundamentals. Build a basic RAG pipeline — PDF ingestion → embeddings → retrieval → answer.
Obs Instrument a Python app with OTel. Add Prometheus metrics + Grafana alerts. Read first 3 chapters of your Observability book.
Claude Agentic Architecture domain deep dive (27% of exam). Cover tool use, multi-turn agents, memory patterns. Target: 200 practice Qs done total.
M3
🏆 First Milestone
Month 3 · Weeks 9–12
Cert + Cloud Starts
Claude Take Claude Certified Architect exam by Week 12. Complete all 500 practice questions. Review Claude Code domain (20% weight) this month.
Python Publish a complete Python app on GitHub with README, tests, CI. Can be a tool, scraper, or mini SaaS backend. Medium post about building it.
AI Start Agentic AI v2. Build a multi-agent system with tool use (web search + file I/O). Understand agent orchestration patterns.
DevOps Start Naina Academy DevOps. Set up GitHub Actions CI/CD pipeline. Containerize your Python app. Docker fundamentals solid.

🔗 Integration Phase
Month 4 · Weeks 13–16
Connect the Dots

DevOps Kubernetes basics. Deploy a containerized app to a managed K8s cluster (AKS or GKE). Start Azure cert prep (AZ-900 or AZ-204).
Python Python + AI integration project: build an agentic tool in Python that uses Claude API. Ship it. Write about it.
Brand 16 posts. Start a mini-series: "How I built X with AI + Python." Each post drives GitHub traffic.

Brand 20 posts. Share cloud cert win. Case study post: "My AI + Observability Stack." Recruiters finding you by now.
