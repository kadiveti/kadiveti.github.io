Project 01
LLM Observability Dashboard
This project instruments LLM API calls using OpenTelemetry spans, tracking every request's latency, token consumption, errors, and cost. Integrated with Grafana for visualizing KPIs and SLO-style alerts.
  • Supports multi-model/multi-provider routing for cost and latency optimization.
  • Automated hallucination monitoring using Langfuse evaluation metrics.
  • Trend analysis of request patterns, performance, and reliability over time.
Tech stack and infrastructure are designed for scalable production deployments.
